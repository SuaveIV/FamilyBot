Setup Help
Bot Token

Get your token at discord.com/developers/applications → Bot → Reset Token.

Admin ID

Enable Developer Mode in Discord → right-click your profile → Copy ID.

Steamworks API Key

Get one at steamcommunity.com/dev/apikey.

Family Group ID

Use steamapi.xpaw.me → IFamilyGroupsService/GetFamilyGroupForUser.

ITAD API Key

Register at isthereanydeal.com.

  1. Open Discord Settings → Advanced → enable Developer Mode.
  2. Right-click any channel or user.
  3. Select Copy ID.
